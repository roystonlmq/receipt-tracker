# Product Overview

Receipts tracking app for managing screenshots and receipts throughout the day. Users can upload, view, rename, and organize screenshots into folders with date-based naming conventions (DDMMYYYY format).

## Core Features

- Upload and store screenshots persistently in a database
- View and rename screenshots
- File explorer with folder navigation
- Auto-upload screenshots based on DDMMYY prefix
- Multi-user profiles with isolated screenshot storage
- Screenshot naming format: "DDMMYY - HHMM - screenshot name.png"

## Use Case

Acts as a photo album classified by folders with dates, allowing users to review and reorganize screenshots at the end of the day (e.g., save to specific project folders).
